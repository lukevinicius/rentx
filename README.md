# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/lukevinicius/rentx.git
```

```bash
# Install Dependencies
$ yarn
# Run Aplication
$ yarn dev
```
Go to http://localhost:3333/ to see the result.
